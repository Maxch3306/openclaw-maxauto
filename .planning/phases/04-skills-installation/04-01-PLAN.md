   - In finally: set installingSkill back to null

5. Pass `installingSkill` and `handleInstall` to `SkillCard` -- add new props:

   - `onInstall: (skill: SkillStatusEntry) => void`

6. In the `SkillCard` component, add the install button in the compact card header row, between the status badge and the ToggleSwitch. The logic:
   - If `canInstallSkill(skill)` is true AND this skill IS the one installing (`installingSkill === skill.skillKey`): show `<Loader2 size={12} className="animate-spin" />` with "Installing..." text (text-[10px], text-[var(--color-text-muted)]).

   - If `canInstallSkill(skill)` is false: show nothing (no button, no disabled button, per user decision).
